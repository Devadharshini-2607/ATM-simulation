# ATM-simulation
This is a simple ATM Simulation Project built using Python.

The program allows users to:
Login using username and PIN
Check account balance
Deposit money
Withdraw money
Exit safely

It is a beginner-friendly project to practice:
Python basics
Dictionaries
Loops
Conditional statements
User input handling

Features
User Authentication
Balance Checking
Deposit Function
Withdraw Function
Insufficient Balance Handling
Menu-driven Program
